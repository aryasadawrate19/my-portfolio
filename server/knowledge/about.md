# About Me

I am a student developer and AI/ML enthusiast focused on building intelligent software systems that combine strong engineering foundations with practical machine learning applications.

My work sits at the intersection of artificial intelligence, developer tools, and system design. I am particularly interested in building systems where machine learning models interact with structured reasoning, operating systems, and real-world workflows.

I have worked on projects involving large language models, retrieval-augmented generation (RAG) systems, semantic search pipelines, and AI-powered developer environments. My projects often explore how AI can be integrated into everyday software systems rather than treated as isolated models.

One of my notable projects is **AInux**, an experimental AI-native Linux environment designed to integrate large language models directly into system workflows. The project explores ideas such as natural-language system interaction, AI-assisted shell commands, and persistent contextual memory within an operating system.

I also build intelligent document understanding systems, including LLM-powered query and retrieval platforms capable of processing PDFs, emails, and structured enterprise documents. These systems focus on extracting relevant information, matching clauses, and generating structured explanations.

Beyond AI systems, I develop full-stack web applications using modern tools such as React, TypeScript, and TailwindCSS. I place strong emphasis on clean architecture, modular components, and responsive user interfaces that remain performant and maintainable.

## My Interests

- Machine learning systems engineering  
- Retrieval-augmented generation and knowledge systems  
- Neuro-symbolic reasoning approaches in AI  
- Developer tools and AI-assisted environments  
- Human–AI interaction and intelligent interfaces  

I am motivated by building software that not only demonstrates technical capability but also explores how intelligent systems can augment real-world workflows and developer productivity.
