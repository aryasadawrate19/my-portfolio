# Experience

My experience primarily comes from building and experimenting with real-world software systems and applied AI projects.

I work across the full stack, designing both user-facing interfaces and backend systems that power intelligent applications. Many of my projects involve integrating machine learning models with practical software environments such as web applications, operating systems, and embedded devices.

## AI and Machine Learning Systems

I have experience building applied AI systems that combine machine learning models with real-world software workflows.

This includes:

- Developing computer vision applications using PyTorch and deep learning models
- Building retrieval-augmented generation (RAG) systems for document understanding and intelligent assistants
- Integrating large language models such as Google Gemini into applications using frameworks like LangChain
- Designing vector database pipelines using tools such as FAISS and Weaviate

These projects focus on making machine learning models usable within larger software systems rather than treating them as standalone experiments.

## Intelligent Software and System Design

I have worked on several system-level and AI-integrated projects that explore how intelligent agents can interact with software environments.

Examples include:

- Designing **AINux**, an experimental AI-native development environment where an intelligent agent can interpret developer intent and automate system workflows
- Building AI-powered tools that combine natural language interfaces with structured retrieval systems
- Experimenting with high-performance inter-process communication and modular system architectures

## Full-Stack Development

Alongside AI systems, I develop full-stack web applications that provide clean and responsive user interfaces for complex backend functionality.

My work typically involves:

- React and TypeScript for frontend development
- Node.js-based backend APIs
- API integration and structured data pipelines
- Designing modular and maintainable application architecture

Across my projects, I focus on building systems that are technically robust, easy to extend, and capable of integrating intelligent behavior into real software environments.
